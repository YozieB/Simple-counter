# Simple Counter
#### ✨Простой счётчик на чистом JavaScript✨

Мини-проект для тренировки
## Реализация

- Метод forEach()
- Конструкция Switch
- Конструкция If / Else
- Свойство classList

## Интересности

- Смена цвета при положительных и отрицательных значениях
- Смена цвета происходит не по клику, а по значению счётчика
